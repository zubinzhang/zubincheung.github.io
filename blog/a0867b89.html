Jenkins配置NodeJS环境

2019-04-04

Jenkins 容器中只有 Java 环境支持运行 jenkins，可通过安装NodeJS 插件提供 Node 运行环境。

下载安装

在 系统管理–>管理插件 下载 NodeJS 插件

使用

  • 进入 系统管理–>全局工具配置–>NodeJS 安装 ，可安装多个 NodeJS 版本

  • 构建环境勾选 Provide Node & npm bin/folder to PATH
